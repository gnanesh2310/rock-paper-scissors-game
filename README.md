# Rock Paper Scissors Game

A classic rock-paper-scissors game implemented in JavaScript.

## Description
A fun and interactive rock-paper-scissors game where you can play against the computer. Built with pure JavaScript, HTML, and CSS.

## Tech Stack
- JavaScript (ES6+)
- HTML5
- CSS3

## Features
- Play against computer AI
- Score tracking
- Move history
- Round counter
- Best of series option
- Reset functionality
- Responsive design
- Clean, intuitive UI

## Getting Started

### Prerequisites
- A modern web browser

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/gnanesh2310/rock-paper-scissors-game.git
   ```
2. Open `index.html` in your browser

## How to Play
1. Open the game in your browser
2. Choose Rock, Paper, or Scissors
3. Computer makes its choice
4. Winner is determined by standard game rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
5. Score is tracked automatically
6. Click Reset to start a new game

## Game Rules
- **Rock** defeats Scissors
- **Scissors** defeats Paper
- **Paper** defeats Rock
- Same choice = Tie

## Features
- **Score Tracking** - Keep track of wins, losses, and ties
- **Round Counter** - See which round you're on
- **Move Display** - See what both players chose
- **AI Logic** - Computer makes strategic choices
